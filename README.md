# Student-Management-System-JDBC

### Project Overview
This is a backend Java application designed to streamline the management of student records. The project focuses on robust database integration using JDBC and adheres to professional software standards, including secure data handling and object-oriented design.

---

### Core Functionality
* **Relational Data Management:** Full implementation of CRUD (Create, Read, Update, Delete) operations.
* **Database Integration:** Utilizes JDBC API for persistent storage with a MySQL backend.
* **System Integrity:** Integrated server-side validation logic to maintain data accuracy for primary keys and student metrics.
* **Architectural Standards:** Developed using Object-Oriented Programming (OOP) principles, specifically Encapsulation for data security and Abstraction for modularity.

### Technical Stack
* **Language:** Java (J2SE)
* **Database:** MySQL
* **API:** JDBC (Java Database Connectivity)
* **Version Control:** Git / GitHub

---

### Setup and Installation
1. **Repository Setup:** Clone the project files to your local workstation.
2. **IDE Configuration:** Import the source files into a Java IDE (IntelliJ IDEA, Eclipse, or VS Code).
3. **Database Configuration:** Execute the provided SQL schema in your MySQL workbench.
4. **Environment Variables:** Update the database connection string (URL, Username, Password) within the source code.
5. **Execution:** Compile and run the application main class.
